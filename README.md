### ◼ [E-Commerce Orders Analysis](https://github.com/amanupa786/ecommerce-orders-analysis)

> **7-phase** pipeline fixing **10 real-world data problems** — charts embedded in Excel + automated email delivery

```
10 Problems Fixed:
 ├─ Mixed date formats (DD/MM/YYYY + YYYY-MM-DD in same column)
 ├─ Currency symbols blocking numeric parse ($1200 stored as text)
 ├─ Discount scale bug (0.10 vs 10.0 for the same 10%)
 ├─ 5 country code variants → 3 clean names
 ├─ Negative quantities corrected with .abs()
 ├─ Invalid emails flagged with str.contains("@")
 ├─ Delivery date before order date → corrected to NaT
 ├─ Duplicate order IDs removed
 ├─ Nulls across 4 columns — 3 different strategies applied
 └─ Inconsistent casing across all text columns standardised

 Output → 3 Matplotlib charts embedded in Excel + HTML email via yagmail
```
